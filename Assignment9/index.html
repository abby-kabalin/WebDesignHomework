How To Build a Working Website

Instructable

Tools

  • Text Editor
    • VS Code
    • IntelliJ
    • Sublime Text

  • Web Browser
    • Google Chrome
    • Windows Explorer
    • Opera

Step 1

Open your text editor and create a new file

  • Save your file as index.html
  • Start your file with the following code:
<!DOCTYPE html>
<html>
<head>
<title>Your Title Here</title>
</head>
<body>
<h1>Your Content Here</h1>
</body>
</html>

Step 2

Fill in your title and content

  • Change Your Title Here to the title of your website
  • Change Your Content Here to the content of your website
<!DOCTYPE html>
<html>
<head>
<title>How to Build a Website</title>
</head>
<body>
<h1>How to Build a Website</h1>
</body>
</html>

Step 3

Save your file and open it in your web browser

  • Right click on your file and select open with
  • Choose your web browser

Step 4

Make changes to your file and refresh your browser to see the changes

  • Change the content in your file
  • Save your file
  • Refresh your browser
